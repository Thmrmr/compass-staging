# COMPASS Staging

HUMAIN COMPASS V2 — Staging Environment  
React + Vite + Supabase (live backend)

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect to Netlify → "New site from Git"
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Deploy

Or just drag the `dist/` folder to [Netlify Drop](https://app.netlify.com/drop)

## Credentials

Same as production COMPASS — connects to the same Supabase project.
